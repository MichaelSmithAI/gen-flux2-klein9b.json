## Runtime downloads and caching

This image downloads models at container startup to avoid long build times. For
Runpod serverless, mount a persistent volume so the models are cached between
cold starts.

Supported environment variables:

- `HF_TOKEN` (required): Hugging Face token for the gated Flux 2 Klein model.
- `LORA_URLS` (optional): Space or newline separated LoRA URLs to download.
- `COMFY_AUTO_UPDATE` (optional): Set to `1` to update ComfyUI on startup.
